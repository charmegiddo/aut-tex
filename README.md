# AUTO-TEX for OSX
About: Automatically tex (include bibtex) compile

How to use:

1) Download

``cd ~;curl -O https://raw.githubusercontent.com/hshintaro575/aut-tex/master/auto-tex.sh;chmod 777 ~/auto-tex.sh``

2) Set alias

``echo -e "\nalias tx='~/auto-tex.sh'"  >> ~/.bashrc``

3) Reload

``source ~/.bashrc``
